# Localized Weather Hazard Reporting and Monitoring System (PSHS-CRC)

![Project Status: WIP](https://img.shields.io/badge/status-work--in--progress-orange)
![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)

This project is a web-based platform designed to enhance campus safety at the **Philippine Science High School - Caraga Region Campus (PSHS-CRC)** in Butuan City. It integrates real-time weather data with community-driven hazard reporting to provide localized situational awareness and actionable safety protocols based on a **Community-Based Early Warning System (CBEWS)** framework.
