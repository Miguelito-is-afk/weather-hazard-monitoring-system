# Localized Weather Hazard Reporting and Monitoring System (PSHS-CRC)

![Project Status: WIP](https://img.shields.io/badge/status-work--in--progress-orange)
![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)

This project is a web-based platform designed to enhance campus safety at the **Philippine Science High School - Caraga Region Campus (PSHS-CRC)** in Butuan City. It integrates real-time weather data with community-driven hazard reporting to provide localized situational awareness.

## ⚖️ Legal Disclaimer & Research Context
**Important Notice:** This system is developed for **academic and research purposes only**. 
1. **Experimental Nature:** As this is a prototype, the data provided may be inaccurate or delayed. 
2. **Not for Official Use:** This platform should **NOT** be used as the primary source for life-safety decisions. Always refer to [PAGASA](https://www.pagasa.dost.gov.ph/) for official weather warnings and local school administration for campus protocols.
